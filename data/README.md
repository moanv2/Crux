# Data — Diego's role

A reproducible, scripted YOLO dataset pipeline. Every step is a standalone,
re-runnable script under `scripts/`, driven entirely by `../config.yaml`
(no hardcoded paths).

## Locked decisions
- **Classes (6, order matters):** `jug, pocket, pinch, sloper, crimp, volume` — see [annotation_guide.md](annotation_guide.md).
- **Approach:** manual, multi-class by hold type. Boxes seeded from a fork of the
  [GDSCMoonless](https://universe.roboflow.com/gdscmoonless/rock-climbing-hold-detection)
  set and **re-typed**, plus the team's own wall photos.
- **Model (downstream):** YOLO26. **Image size:** 640.

## Run the whole thing

```bash
pip install -r ../requirements.txt
$env:ROBOFLOW_API_KEY="..."          # PowerShell (bash: export ROBOFLOW_API_KEY=...)
python scripts/run_pipeline.py        # download -> clean -> ingest -> stats -> split -> export -> QA
```

Or run any step on its own, e.g. `python scripts/sanity_check.py`.

## Steps

| Script | Status | Does |
|---|---|---|
| `download.py` | ✅ | Pull our annotated Roboflow version via the SDK into `raw/`. |
| `clean.py` | 🟡 optional | Dedupe / drop junk. Resize+EXIF can be done in Roboflow at version-gen instead. |
| `ingest_custom.py` | ⚪ optional | Only needed if photos are annotated *outside* the main Roboflow project. |
| `stats.py` | ✅ | Profile class balance + counts → `data_card.md`. |
| `split.py` | ✅ | Train/val/test split (seeded, no leakage) → `processed/`. Stratify = TODO. |
| `export.py` | ✅ | Write `data.yaml` for Ultralytics. |
| `sanity_check.py` | ✅ | Label QA: per-class counts, empty/oversized/degenerate boxes. |

`config.py` / `labels.py` are shared helpers (config loader, YOLO label IO).

> **Roboflow vs scripts:** a downloaded Roboflow *version* already comes resized,
> (optionally) split, and with a `data.yaml`. Decide once: let Roboflow do
> preprocessing/split and use our scripts mainly for the reproducible **pull + QA +
> data card**, *or* export unsplit and let `clean/split/export` own it. Don't do both.

## Layout

```
raw/        Roboflow download (gitignored)
custom/     team phone photos + their Roboflow export (custom/incoming/)
interim/    cleaned + merged, flat images/ + labels/
processed/  final train/val/test YOLO dataset
data.yaml   generated by export.py  ← handoff to Coder 1
data_card.md generated by stats.py
annotation_guide.md  the hold-type rubric (read before annotating)
```

## Handoff to Coder 1
`data.yaml` + `data_card.md` (with the Roboflow version ID recorded).
